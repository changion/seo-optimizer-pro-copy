  SEO Optimizer Pro - 开始使用指南
═══════════════════════════════════════════════════════════

✅ 项目已完成！所有代码和配置文件已准备就绪。

📋 接下来您需要做的：

1️⃣  安装 Node.js（如果还没有安装）
    - 访问：https://nodejs.org/
    - 下载并安装 LTS 版本
    - 详细说明：查看 INSTALL_NODEJS.md

2️⃣  安装后端依赖
    方法A - 使用安装脚本（推荐）：
    cd server
    ./install.sh          # Mac/Linux
    或
    install.bat           # Windows

    方法B - 手动安装：
    cd server
    npm install

3️⃣  启动后端服务器
    cd server
    npm run dev

4️⃣  启动前端服务器（新终端窗口）
    cd /Users/chenjunrun/Documents/txt
    python3 -m http.server 8080

5️⃣  访问网站测试
    打开浏览器访问：http://localhost:8080/seo-analyzer.html
    输入网站URL（如：https://example.com）
    点击 "Analyze SEO" 查看结果！
