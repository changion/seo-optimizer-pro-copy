# SEO Optimizer Pro - SEO优化平台

一个专为美国市场设计的AI驱动SEO优化平台，帮助企业和营销人员提升搜索引擎排名、增加有机流量。

## 🌟 项目定位

**SEO Optimizer Pro** 是一个专注于SEO优化的SaaS平台，提供：

- 🔎 **关键词研究工具**：搜索量、竞争度、难度分析
- 📄 **页面SEO分析器**：全面的页面SEO审计和建议
- 📝 **内容优化工具**：AI驱动的SEO内容优化
- 🏷️ **Meta标签生成器**：优化的标题和描述标签
- 🔗 **外链分析器**：外链质量分析和建设机会
- 📊 **SEO评分检查器**：综合SEO评分和改进建议

## ⚙️ 配置说明

### 时区设置

默认时区设置为 `America/New_York`（美国东部时间）。如需更改，请编辑 `script.js` 文件：

```javascript
const US_CONFIG = {
    timezone: 'America/New_York', // 可改为其他美国时区
    // ...
};
```

**常用美国时区选项**：
- `America/New_York` - 东部时间 (EST/EDT)
- `America/Chicago` - 中部时间 (CST/CDT)
- `America/Denver` - 山地时间 (MST/MDT)
- `America/Los_Angeles` - 太平洋时间 (PST/PDT)

### 区域设置

所有区域相关配置都在 `script.js` 的 `US_CONFIG` 对象中：

- **locale**: `en-US` - 美式英语
- **currency**: `USD` - 美元
- **dateFormat**: 美式日期格式
- **timeFormat**: 12小时制时间格式
