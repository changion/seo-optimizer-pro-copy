## 🎯 操作步骤

### 第一步：安装 Node.js

您的系统上未检测到 Node.js，需要先安装。

#### Mac 用户（推荐使用 Homebrew）

```bash
# 如果已安装 Homebrew
brew install node

# 验证安装
node --version
npm --version
```

#### 所有系统用户

1. 访问 Node.js 官网：https://nodejs.org/
2. 下载 LTS（长期支持）版本
3. 运行安装程序
4. **重新打开终端**后验证：
   ```bash
   node --version
   npm --version
   ```

详细说明请查看：`INSTALL_NODEJS.md`

### 第二步：安装后端依赖

安装 Node.js 后，运行安装脚本：

#### Mac/Linux 用户

```bash
cd server
./install.sh
```

或手动执行：

```bash
cd server
npm install
```

#### Windows 用户

```bash
cd server
install.bat
```

或手动执行：

```bash
cd server
npm install
```

### 第三步：启动后端服务器

```bash
cd server
npm run dev
```

看到以下消息说明启动成功：
```
🚀 SEO Optimizer Pro API server running on port 3000
📡 CORS enabled for: http://localhost:8080
🌍 Environment: development
```

**保持这个终端窗口打开！**

### 第四步：启动前端服务器

打开**新的终端窗口**：

```bash
cd /Users/chenjunrun/Documents/txt
python3 -m http.server 8080
```

### 第五步：测试系统

1. 访问：`http://localhost:8080`
2. 点击 "Start Free Analysis" 或访问：`http://localhost:8080/seo-analyzer.html`
3. 输入网站URL（例如：`https://example.com`）
4. 点击 "Analyze SEO" 查看结果！

## 🐛 遇到问题？

### 问题：找不到 node 命令

**解决方案**：
1. 确认 Node.js 已安装
2. **重新打开终端**（环境变量需要重新加载）
3. 检查 PATH 环境变量

### 问题：npm install 失败

**解决方案**：
```bash
# 清除缓存
npm cache clean --force

# 使用国内镜像（中国用户）
npm config set registry https://registry.npmmirror.com

# 重新安装
npm install
```

### 问题：端口被占用

**解决方案**：
- 修改 `server/.env` 中的 `PORT=3000` 为其他端口（如 `PORT=3001`）
- 同时更新 `seo-analyzer.js` 中的 `API_BASE_URL`
