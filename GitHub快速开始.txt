  GitHub 快速开始指南
═══════════════════════════════════════════════════════════

📋 您需要做的3件事：

1️⃣  创建GitHub账号和仓库
   ──────────────────────────────────────────────
   • 访问：https://github.com
   • 注册账号（免费）
   • 创建新仓库（Repository）
   • 仓库名：seo-optimizer-pro
   • 选择：Public（公开）

2️⃣  上传代码到GitHub
   ──────────────────────────────────────────────
   
   方法A：使用脚本（最简单）✨
   ────────────────────────────
   cd /Users/chenjunrun/Documents/txt
   ./一键上传到GitHub.sh
   
   方法B：使用GitHub Desktop（图形界面）✨
   ────────────────────────────────────────
   • 下载：https://desktop.github.com
   • 安装并登录
   • 添加本地仓库
   • 发布到GitHub
   
   方法C：使用命令行
   ──────────────────
   cd /Users/chenjunrun/Documents/txt
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/您的用户名/seo-optimizer-pro.git
   git push -u origin main

3️⃣  连接部署平台
   ──────────────────────────────────────────────
   • Vercel（前端）：https://vercel.com
   • Railway（后端）：https://railway.app
   • 都用GitHub账号登录
   • 选择您的仓库
   • 自动部署！
