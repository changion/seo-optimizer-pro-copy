### 停止服务器

如果需要停止服务器：

**停止后端**：
```bash
# 查找进程
lsof -ti:3000

# 停止进程（替换PID为实际进程ID）
kill -9 <PID>
```

**停止前端**：
```bash
# 查找进程
lsof -ti:8080

# 停止进程
kill -9 <PID>
```

## 💡 提示

- 后端服务器在后台运行，保持终端打开
- 前端服务器也在后台运行
- 如果遇到问题，检查浏览器控制台和终端日志
- 某些网站可能会阻止爬虫，这是正常现象

## 🐛 故障排除

### 问题：无法访问前端
- 检查前端服务器是否运行：`lsof -ti:8080`
- 确认端口8080未被占用

### 问题：SEO分析失败
- 检查后端服务器是否运行：访问 http://localhost:3000/api/health
- 查看后端终端日志了解错误信息
- 某些网站可能阻止爬虫访问

### 问题：CORS错误
- 检查 `server/.env` 文件中的 `CORS_ORIGIN` 设置
- 确保与前端地址匹配（http://localhost:8080）
